The Drcom-Client Open Source Project

Welcome to the website for the Drcom-Client Open Source Project!


Drcom-Client is an open source ISP client for logging in to the 2133 Broadband Remote Access Server (also known as Dr.COM, simplified Chinese: 计费网关). It is released under the GNU General Public License. It is available for GNU/Linux, Microsoft Windows now.
