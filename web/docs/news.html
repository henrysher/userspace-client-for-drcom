2009-05-27: drcom-client.ORG Launched!


A big THANKS to all our supporters for their LOVE to drcom-client!

We have finally got drcom-client.org Online!

2009-05-25: drcom-client PUM v1.0 released

Support GNU/Linux

New features:
  • Add friendly Graphical User Interface for esay configuration
  • Support present Dr.COM protocols, except for v3.72_u31_2227
  • Handle except address more accuracy
  • Add multi-language support to message display
  • Rewrite user-mode protocol by Python

Support GNU/Linux

  • Add README in English
  • Fix: acquire all except addresses from server_ack

2008-11-02: drcom-client v1.4.8.1 released

Support GNU/Linux

2008-10-22: drcom-client v1.4.8 released

Support GNU/Linux

2008-05-17: drcom-client v1.4.6 released

Support GNU/Linux

2008-04-12: drcom-client v1.4.4 released

2008-03-31: drcom-client v1.4.0 released

Support GNU/Linux

New features:
  • Add netfilter-based kernel module to solve the problem of TCP/UDP header prefix authentication
  • Rewrite the whole framework from drcom-client v1.3.7
Bugfixes and minor changes:
  • Use pthread_cancel(), to avoid memory leak
  • Replace old md5.c/md5.h with new one
