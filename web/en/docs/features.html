Drcom-Client Features

Overview


Drcom-Client is a reliable cross-platform ISP client with lots of useful features and an intuitive graphical user interface.

Features


Among others, the features of Drcom-Client include the following:

  • Easy to use
  • Easy to install

    • Users could choose source code based installation or binary-package(.deb, .rpm, etc.) based installation

  • Friendly graphical user interface for easy configuration

    • Provides 100% GUI configurability, no need to edit configuration files by hand and restart the program.

  • Supports almost present Dr.COM Protocols

    • Protocol compatiable with Dr.COM Client v3.46, v3.48, v3.50, v3.71, v3.72, except for v3.72_u31_2227_build

  • Cross-platform

    • Already available on GNU/Linux, and goint to support Microsoft Windows, MacOSX and Other *BSD platforms

  • Available in both Chinese and English
  • For C-L-I fans, still supports command line interface
