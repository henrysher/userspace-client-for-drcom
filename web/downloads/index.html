Unstable release


You can check out the latest unstable release of drcom-client using git. To access a Git repository, configure your Git client as follows: 
git clone git://drcom-client.git.sourceforge.net/gitroot/drcom-client
If you are a developer of drcom-client, and plan on pushing changes back upstream and have a sourceforge account, use the following instead: 
git clone ssh://USER@drcom-client.git.sourceforge.net/gitroot/drcom-client
For more information on this, see the instructions on how to use the repository
