提交 BUGs

你发现了一个Bug? 这里有5个步骤，可以将它修正!


  1. 检查你的版本

    为了保证不浪费大家的时间, 在你提交你的Bug之前最好使用最新版本的。首先在 这里 下载最新的 drcom-client。如果你已经做了这一步，还是遇到这个Bug，那么请你看下一步。


  2. Bug Tickets中搜索

    现在你已经使用了最新版本的 drcom-client 并且仍然认为你发现了一个Bug。那么请你先在 Tickets 中搜索，确认你是否是第一个发现这个问题的。这一步非常重要。如果你发现已经有人将类似的Bug提交，那么你就不需要再重新提交了，只要在该Bug之后写上你的评论即可。如果你在Tickets中没有发现类似的Bug，那么继续下一步。


  3. 提交你的Bug

    现在你准备向我们 提交Bug 了。 文档 如何写出好的提交报告 介绍了一些有用的Bug描述技巧。你的提交报告越出色，你的Bug被接受和修正的所需要的时间会越短。请认真撰写你的提交报告。


  4. 然后呢?

    一旦你的Bug提交之后，当该Bug的处理更新到新的阶段时，你将收到一份通知邮件。具体请见Bug生命周期。 在Bug被认为已经修正之后，你可以下载最新的版本来确认改Bug是否被修正。
