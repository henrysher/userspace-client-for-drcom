Drcom-Client 开源项目

Drcom-Client 子项目


drcom-client 有一些有意思的子项目，我们正在积极努力开发。想了解更多有关各个子项目的内容，你可以直接访问该子项目的主页。如果你有兴趣，开始新的子项目，请联系我们!

基本项目

  1. drcom-client (基于 GNU/Linux)

    2. 该项目针对 GNU/Linux 平台。我们的开发者主要聚集在这个项目上面，原因是城市热点公司，没有发布基于 Linux 系统的官方 Dr.COM® 客户端。 目前我们已经发布了 drcom-client v 1.4.8.2。 具体请访问该项目主页。


  2. drcom-client (基于 Microsoft Windows)

    3. 开发 Windows 版本的 drcom-client， 其目标是帮助用户通过 NAT/代理服务器的方式共享接入互联网，而这一点恰恰是官方 Dr.COM® 客户端所限制的。 具体请访问该项目的首页。


  3. drcom-client (基于 Mac OS X)

    4. Mac OS X 版本是在另一个子项目 drcom-client PUM 的基础上进行开发的。目前还处于 pre-alpha 阶段。 具体请访问这里。

特别项目

drcom-client PUM
drcom-client PUM是 基于GNU/Linux drcom-client 项目的一个子项目。这个子项目的目标是创建一个带有图形界面且易于配置的用户态客户端。PUM 是 Python 用户态的缩写。由于 Python 在原型机设计中的强大功能，我们将它作为该子项目的主要编程语言。最近我们已发布了 drcom-client PUM v1.0。
