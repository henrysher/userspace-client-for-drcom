不稳定版本


您可以利用git 客户端查看最新的drcom-client不稳定版本。要访问drcom-client源码库，使用你的git 客户端： 
git clone git://drcom-client.git.sourceforge.net/gitroot/drcom-client
如果你是drcom-client的开发者，准备对代码做些修改，首先请申请一个 SourceForge 帐号，然后进行如下操作: 
git clone ssh://USER@drcom-client.git.sourceforge.net/gitroot/drcom-client
更多的信息，请阅读如何使用drcom-client源码库。
