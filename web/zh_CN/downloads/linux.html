To install drcom-client you may find the Tutorials helpful. For additional help, the FAQ is a good starting point.

Dependencies


drcom-client PUM v1.0
  • GNOME Desktop required
  • Python >=2.4 (not include 3k) required
  • PyGTK >=2.0 required

Binaries

Sources


Grab the latest version of the source from below and have fun compiling!

drcom-client PUM v1.0
drcom-client v1.4.8.2

You might want to compare the MD5 sum to verify the integrity of downloaded packages.

Note:
Ubuntu users might want to install the dkms package (not available on Debian) to ensure that the drcom-client kernel module (drcom) are properly updated if the linux kernel version changes during the next apt-get upgrade.
