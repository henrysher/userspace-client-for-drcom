Drcom-Client 开源项目

下载 drcom-client (基于GNU/Linux)

当前稳定版本

  1. 图形界面: drcom-client PUM v1.0
  2. 命令行: drcom-client v1.4.8.2

安装 drcom-client，请仔细阅读安装教程。如果有其它的问题可以参考常见问题。

依赖关系


drcom-client PUM v1.0
  • Linux 内核版本 >= 2.6.24
  • 桌面系统: GNOME 2
  • Python >=2.4 (不包含Python 3)
  • PyGTK >=2.0

二进制安装包


基于GNU/Linux 的 drcom-client 二进制安装包，根据不同的 Linux 发行版，目前支持的类型:

现在只适用于drcom-client PUM v1.0

Ubuntu/Debian Package (.deb)
Fedora Package (.rpm)
注意:
上述列表目前还不完整，如果你所使用的 Linux 发行版无法使用上述二进制安装包，我们推荐从源码来编译和安装 drcom-client 。同时你也可以和我们联系，我们将尽快支持你所使用的 Linux 发行版。

源代码


下载最新的源码包，体验编译的乐趣!

drcom-client PUM v1.0
drcom-client v1.4.8.2

你可能想要源码包的MD5信息，来验证下载的完整性。

注意:
Ubuntu 用户可能需要安装 dkms 包 (目前在 Debian 系统下还没有得到支持)。这个包的作用是保证 drcom-client 认证模块(drcom) 能随着 Linux 内核的改变而及时地编译安装。
